## Installation and Usage <a name="installation-usage"></a>
- Requirements
  - `java >= 1.8`
  - `javafx`
  - `aws account` - for running AWS-RDS service (optional)
- `git clone https://github.com/AI-14/kfupm-clubzz.git` - clones the repository
- `cd kfupm-clubzz`
